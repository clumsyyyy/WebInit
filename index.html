Solving The Digital Talent Issues

“The Initiator” adalah sebuah Aksi Angkatan dari INIT 2020 sebagai langkah konkret untuk memperkenalkan teknologi digital dalam rangka mendukung pertumbuhan talenta digital di Indonesia, terkhusus mahasiswa dan pelajar.

Init hadir di tengah masyarakat untuk memberikan pengetahuan umum mengenai teknologi. Init berinisiatif untuk memperkenalkan teknologi dan mendukung pertumbuhan talenta digital dalam bentuk talkshow, bootcamp, dan konten edukatif melalui media sosial.

Programming Bootcamp: Intro to Python

Acara pelatihan berbentuk bootcamp dengan total 3 pertemuan. Untuk setiap pertemuan, terdapat 2 jam untuk mengenalkan bahasa pemrograman Python kepada masyarakat, terutama mahasiswa TPB untuk mempersiapkan mata kuliah Pengenalan Komputasi.

Pengenalan Python ini bertujuan untuk mengajarkan dasar-dasar Python dan cara berpikir komputasional menggunakan Python. Bootcamp ini akan berlangsung selama 3 hari, yaitu pada tanggal 13, 15, dan 16 Agustus. Materi yang akan disampaikan pada bootcamp pengenalan Python ini adalah data structure (int, str, list), variabel, operator (+, -, hingga modulo), I/O, control flow (if-else statement, while loop, for loop), dan fungsi di Python. Akan ada soal-soal permasalahan yang akan membantu kalian memahami Python serta mengembangkan computational thinking kalian.
